Keep the Algorithm Doc Current Skill

This skill provides:
- SKILL.md: the policy requiring docs/ALGORITHM.md updates for any exploration or change
  to the download/ad-cutting process (including failed experiments)
- skill.json: metadata and pointers

docs/ALGORITHM.md is the research & validation history behind the ad-cutting algorithm —
the serving model, disproven approaches, invariants, measured results and the validation
playbook. It exists so future work doesn't re-lose expensively-gained knowledge.
